# node-server-demo
完成一个提交及图片展示的demo，初步了解node的开发机制。

> node.js 入门

《Node入门》，地址：[https://www.nodebeginner.org/index-zh-cn.html](https://www.nodebeginner.org/index-zh-cn.html)

## 项目运行：

命令行窗口，开启本地后台node服务器：

```
#进入项目目录
cd node-server-demo
    
安装依赖包
cnpm install --save
    
运行    
node index.js
```
浏览器打开：

```
localhost:8888
```
